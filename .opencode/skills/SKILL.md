# MouseAI — Skills Disponíveis

Este diretório contém skills especializados para trabalhar no projeto MouseAI.

## Skills

| Skill | Descrição |
|---|---|
| **[vps-port-forwarding](vps-port-forwarding/)** | Como expor os containers da VPS via SSH port forwarding |
| **[mcp-config](mcp-config/)** | Como configurar MCPs (GitHub, PostgreSQL, GitLab, Code Interpreter) no LobeChat |
| **[github-mcp](github-mcp/)** | Como usar o GitHub MCP para ler e analisar repositórios |
| **[ai-performance](ai-performance/)** | Boas práticas e tips de performance para prompts e código |
| **[project-overview](project-overview/)** | Visão geral da arquitetura, estrutura e serviços do MouseAI |

## Uso

Ao trabalhar no projeto MouseAI, carregue o skill relevante com a ferramenta `skill`:

```
skill: vps-port-forwarding   # Configuração de VPS
skill: mcp-config           # Configuração de MCPs
skill: github-mcp           # Análise de repositórios GitHub
skill: ai-performance        # Boas práticas
skill: project-overview     # Visão geral do projeto
```
